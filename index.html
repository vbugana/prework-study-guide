HTML

  • The head element contains information about the webpage.
  • meta defines metadata about an HTML document.
  • The body element defines the document's body visible content shown to the user.
  • The h1 element defines a heading.
  • The p element defines a paragraph.
  • HTML is Not Case Sensitive.
  • !DOCTYPE defines the document type.
  • h1 to h6 defines HTML headings.
  • img defines an image.
  • li defines a list item.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The color Property Is not Just for Text
  • The visibility Property Can Be Set to “collapse”
  • The background Shorthand Property Has New Values
  • The clip Property Works Only on Absolutely Positioned Elements
  • Vertical Percentages are Relative to Container Width, Not Height
  • The border Property is Kind of Like Inception
  • The text-decoration Property is Now a Shorthand
  • The border-width Property Accepts Keyword Values

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • gitclone is a command for downloading existing source code from a remote repository.
  • git branch for creating, listing and deleting branches.
  • git checkout for switching from one branch to another, or for checking out files and commits.
  • git status command gives us all the necessary information about the current branch.
  • git add to include the changes of a file(s) into next commit.
  • git commit is like setting a checkpoint in the development process which you can go back to later if needed.
  • git commit saves your changes only locally.
  • git push uploads your commits to the remote repository.

Javascript

  • JavaScript is an object-oriented programming
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript allows users to interact with web pages.
  • JavaScript can change HTML content.
  • JavaScript can change HTML attributes.
  • JavaScript can change CSS style.
  • JavaScript can hide HTML elements.
  • JavaScript can show hidden HTML elements.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • A for loop that can iterate over the array and log each topic in the console with only a few lines of code.
